Evolving the JT-ware Software Developer's Kit (JTSDK)

Version 4 is On Its Way !

Introduction.

    The Hamlib SDK Developers need to research what happens with regards to Qt 5.15.2 delivery.

    As of early March, 2024 Qt 5.15.2 is now not easily delivered via scripted means via the Qt Installer and/or Qt Maintenance Tool. It is only available from "Archive".

    This is the prompt to actively commence work on Version 4 of the JTSDK.

    The JTSDK Version 4 will be Qt 6.6 (and later -centric.

    Yet the kit will still support Qt 5.15.2  if it is deployed manually from Archive sources. CLick here for a resource that details how to do this from the Qt Maintenance Tool.
