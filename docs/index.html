Important Note.

    Development of the Version 3-Streams of the Hamlib SDK (JTSDK) has concluded. Further maintenance will not be performed on images from this stream.

    If third-parties wish to continue any component of the Version 3-Stream of the Hamlib SDK please contact the developers via the forum. Tested efforts can and will be uploaded and made available here.

Introduction.

    Version 4.0 of the Hamlib SDK (JTSDK) is now current.

    Version 4 of the JTSDK/Hamlib SDK aims to be > Qt 6.6.3 centric - but will support archive-repository-delivered deployments of Qt 5.15.2

    A Version 4.1 Preview is available at https://sourceforge.net/projects/hamlib-sdk/files/Windows/JTSDK-4.1-Stream/. This stream contains several structural improvements. This stream may go mainstream soon.

    Please watch this space and information/notes made available via the jtsdk@groups.io forum.
